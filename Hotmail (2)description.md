## 🚀 Overview
Shadow Realm Hotmail Checker v2 is a multi-threaded Hotmail/Outlook inbox checker that scans accounts for emails from 10+ major platforms including Facebook, Instagram, TikTok, Twitter, LinkedIn, Pinterest, Netflix, Spotify, and more. Valid platform hits are saved to separate categorized files.

---

## 🎮 Features

### 📧 Detection Targets
- Facebook, Instagram, TikTok, Twitter
- LinkedIn, Pinterest, Snapchat
- Netflix, Spotify, Disney+, Hulu
- YouTube and many more
- Optional Telegram bot notifications

### ⚙️ System Features
- ⚡ Multi-threaded high-speed checking
- 📊 Live hit counter per platform
- 📁 Separate hit files per platform
- 🔄 Concurrent.futures thread pool
- 🌍 Country detection per account
